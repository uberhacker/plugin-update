# plugin-update
A simple bash shell script to update all your Terminus plugins

# Installation:
```
$ git clone https://github.com/uberhacker/plugin-update.git
$ cd plugin-update
$ chmod +x plugin-update
$ sudo mv plugin-update /usr/local/bin
```

# Usage:
```
$ plugin-update
```
All your Terminus plugins should now be updated.  Remember to execute this script occasionally to make sure plugins are up to date.
